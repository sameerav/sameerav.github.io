Parallel Computing Laboratory

Research Laboratory Assistant• January 2015 - Present

I built a DSL (domain-specific language) called Grapher that can be used for easy and efficient data visualization and pattern finding within datasets. Grapher should be used through a Django-based web app interface. Given a data set as a CSV or in JSON format, Grapher utilizes d3 to parse and graph the data in a variety of different chart types. Grapher can also normalize data, pattern-fit data, and somewhat intelligently pre-process data based on data types (time, date, count, scaled value, etc).
